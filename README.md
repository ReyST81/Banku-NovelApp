# BANKU ("BAcaaNKU")

Banku adalah sebuah aplikasi bagi penulis untuk mempublikasi karyanya dan juga bagi pembaca untuk membaca novel sepuasnya secara gratis

## Fitur MVP:

- Post Novel
- Read Novel
- Edit Novel
- Delete Novel
- Search Novel

## Screen

- Login Screen
- Register Screen
- Home Screen
- Post Screen
- MyNovel Screen
- Search
